Usage Ascii-code and Visual Crypotography

Use library pillow and pytesseract

``pip install pillow``

``pip install pytesseract``

Usage:
    ``python encode.py``
        - Encode from string to text and to 2 shares
    ``python decode.py``
        - Decode from text to string
